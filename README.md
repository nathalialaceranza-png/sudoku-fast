# Sudoku Fast

**Real-time competitive Sudoku built for one-handed mobile speed.**

Play against players worldwide or an adaptive CPU bot that learns from your
matches. Swipe, long-press, and drag to fill — zero unnecessary taps.

## Features
- 5 difficulty levels (Easy to Master)
- Real-time PvP matchmaking with cross-difficulty fallback
- Adaptive CPU opponent trained on real player times
- Practice mode with personal records
- One-handed mobile-first controls
- Notes/pencil marks, undo, freeze penalty
- Completion wave animations
